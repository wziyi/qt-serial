### qt_serial

这是用 qt 开发的串口小工具  

### 开发环境

Qt 5.14.1 + MinGW 7.3.0

### 使用说明

1、请不要使用 visual studio 编译器。   
2、在有些电脑上存在无法接收数据的现象，暂未找到原因。   
3、代码存在诸多问题，仅供参考。
